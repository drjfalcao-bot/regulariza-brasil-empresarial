# Regulariza Brasil Empresarial

Landing page estática para captação de leads de regularização fiscal, administrativa e regulatória.

## Arquivo principal

`index.html`

## Publicação pelo GitHub Pages

1. Acesse **Settings** do repositório.
2. Entre em **Pages**.
3. Em **Build and deployment**, selecione **Deploy from a branch**.
4. Escolha branch `main` e pasta `/root`.
5. Salve.

URL esperada após ativação:

`https://drjfalcao-bot.github.io/regulariza-brasil-empresarial/`
